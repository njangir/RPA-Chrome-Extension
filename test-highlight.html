Manual Testing

Open browser console and run:

// Test highlighting specific elements
window.testHighlight('#test-button-1')
window.testHighlight('.test-input')
window.testHighlight('a.test-link')

// Test custom selectors (if extension is loaded)
window.testCustomSelector(0, '#test-button-1')
window.testCustomSelector(0, '.test-input')
window.testCustomSelector(0, 'button[data-testid="special-button"]')

// Remove highlight
window.testRemoveHighlight()
